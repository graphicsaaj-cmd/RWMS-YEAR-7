# RWMS Year 7 Business Studies CBT

Computer Based Test (CBT) system for Royal Wunderkind Model School.

## Features
- 45 minute exam timer
- 60 objective questions
- Automatic scoring
- Shows correct answers after submission
- Works offline in any browser

## How to Use
1. Download the repository or ZIP file.
2. Open `index.html` in your browser.
3. Students answer questions and submit to see their score.

## Deployment
You can host this CBT online using:
- GitHub Pages
- Netlify
- School web server
